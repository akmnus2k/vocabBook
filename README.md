# 📘 PT 单词本

给物理治疗（PT）专业学习者的英语单词工具：查词联想 → 看释义/例句/图片 → 收藏 → 间隔复习。

## 功能

- **查单词**：输入前几个字母自动联想候选词（带中文释义预览）；查词结果包含美/英音标、真人发音、中文释义、双语例句和相关图片
- **单词本**：一键收藏，支持导出 CSV 备份
- **复习**：抽认卡模式 + 间隔重复——认识的词复习间隔逐渐拉长（1/2/4/7/15/30 天），不认识的词当天重来

## 日常使用

- **双击桌面的「PT单词本」图标**即可打开（会自动启动后台服务再打开网页）
- 电脑**开机时服务会自动启动**（启动脚本在开始菜单的"启动"文件夹里）
- **每晚 8 点自动检查**：有到期单词就弹窗提醒，点"是"直接打开单词本

手动运行（备用）：

```bash
cd vocabBook
streamlit run app.py
```

### 修改提醒时间

管理员或普通命令行运行（把 20:00 改成想要的时间）：

```
schtasks /change /tn VocabBook-DailyReview /st 20:00
```

## 文件说明

| 文件 | 作用 |
|---|---|
| `app.py` | 界面（三个标签页：查词 / 单词本 / 复习） |
| `dict_api.py` | 词典接口（有道联想、有道词典、必应图片，均为免费接口） |
| `storage.py` | 单词本存储（本地 `vocab_book.json`）和复习计划 |
| `vocab_book.json` | 收藏的单词数据（自动生成，删除即清空单词本） |
| `open_vocab.bat` | 桌面图标指向的启动器：确保服务在运行，然后打开网页 |
| `start_server.vbs` | 无窗口启动后台服务（开机自启用的就是它的副本） |
| `remind.vbs` + `check_due.py` | 每日提醒：查有没有到期单词，有就弹窗（计划任务 VocabBook-DailyReview 每晚 8 点调用） |

## 数据说明

所有收藏数据保存在本地 `vocab_book.json`，不依赖任何账号；查词需要联网。
