# ☁️ 部署到云端（发给她用）

照下面四步做完，你会得到一个网址，发给她就能用——手机、电脑、任何地方（需科学上网）。
代码我已经提交到本地 git 仓库，只差推送和部署。

## 第 1 步：在 GitHub 建一个仓库

1. 打开 https://github.com/new （用你的 akmnus2k 账号登录）
2. Repository name 填 `vocabBook`
3. 选 **Private**（私有，代码不公开）
4. 其他都不勾，点 **Create repository**

## 第 2 步：推送代码

在命令行执行（也可以让 Claude 帮你跑）：

```
cd D:\JProjects\vocabBook
git remote add origin https://github.com/akmnus2k/vocabBook.git
git push -u origin main
```

## 第 3 步：部署到 Streamlit Cloud

1. 打开 https://share.streamlit.io ，点 **Sign in with GitHub** 登录并授权
2. 点右上角 **Create app** → "Deploy a public app from GitHub"
3. 填写：
   - Repository：`akmnus2k/vocabBook`
   - Branch：`main`
   - Main file path：`app.py`
   - App URL：起个好记的名字，比如 `pt-vocab`
4. 点开 **Advanced settings**，在 **Secrets** 框里粘贴
   `D:\JProjects\vocabBook\.streamlit\secrets.toml` 的**全部内容**
   （用记事本打开复制即可——这一步就是把 Google 表格钥匙和访问密码交给云端）
5. 点 **Deploy**，等两三分钟

## 第 4 步：发给她

部署完会得到网址，比如 `https://pt-vocab.streamlit.app`

发给她：**网址 + 访问密码 `pt2026`**

手机上用浏览器打开、输一次密码就能用。建议她用浏览器的
**"添加到主屏幕"** 功能，以后点图标就像打开 App 一样。

## 常见问题

**改访问密码**：改两处——本地 `.streamlit/secrets.toml` 里的 `app_password`，
和 Streamlit Cloud 后台（App → Settings → Secrets）里的同一行。

**数据存在哪？** 你 Google Drive 里那个叫 "App" 的表格，
新增了 `vocab_words`（单词本）和 `search_history`（搜索历史）两个分页。
你电脑上的应用和云端网页读写的是同一份数据。

**国内访问**：`*.streamlit.app` 在国内需要科学上网。她如果没有梯子，
备用方案是你俩在同一 WiFi 时访问你电脑的地址（问 Claude 要局域网方案）。

**改代码后更新云端**：`git add -A`、`git commit -m "说明"`、`git push`，
Streamlit Cloud 会自动重新部署。
